Sketch

I use the program sketch to create a mockup of the website. I absolutely love the versitility of the application. Lots of artboards for any size of screen. It has great vectoring, which is great for creating and exporting icons. 

Pigendo

A great free mockup tool that I have been using lately.  Lets me quickly create a site in bootstrap and be mobile ready in minutes!
